# Commands Executed

```bash
git init
echo '# My cool project' > README.md
echo 'puts "Hello again"' > app.rb

git status                  # -> Untracked files
git add README.md
git status                  # -> Changes to be committed + untracked
git add app.rb
git commit -m "Initial commit with README and app.rb"
git status                  # -> nothing to commit, working tree clean
```
