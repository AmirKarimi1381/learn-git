# Executed Commands

### 1. Create new folder

```bash

mkdir 02-more-setup

cd 02-more-setup

```

### 2. Download with curl

```bash

curl -L -O https://gitimmersion.com/git_tutorial.zip

```

### 3. Extract the file

```bash

unzip git_tutorial.zip

```

### 4. Check contents

```bash

cd git_tutorial

ls -la

cd work

```

### Summary of work:

Created a new folder, downloaded the educational file, extracted it, and checked its contents.
